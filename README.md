# VoltSetu
voltsetu.netlify.app


VoltSetu is a marketplace connecting EV riders with nearby charging spot hosts in Kolhapur, Maharashtra.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool & dev server
- **TypeScript** — Static typing
- **Tailwind CSS** — Utility-first styling
- **shadcn/ui** — Accessible component library
- **Firebase** — Auth (Google Sign-In) + Realtime Database
- **Leaflet** — Interactive maps (Carto Voyager tiles, OSM Nominatim geocoding)
- **Capacitor** — Android app packaging (same codebase, no rewrite)

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Run tests
npm run test

# Build for production (also regenerates public/sitemap.xml automatically)
npm run build
```
